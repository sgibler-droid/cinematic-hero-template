# Cinematic Hero Template

A dependency-free, full-screen website hero with responsive typography, glass controls, an original generated poster, motion, and reduced-motion support.

## Preview

Run `python3 -m http.server 8080` in this directory, then open `http://localhost:8080`.

## Customize

- Replace placeholder brand and copy in `index.html`.
- Replace `assets/hero-cinematic.jpg` with a poster of the same name.
- Add `assets/hero.webm` and/or `assets/hero.mp4`; the poster remains the fallback.
- Change the palette and page spacing in the variables at the top of `styles.css`.

The generated hero art was created with OpenAI's built-in image generation tool from an original prompt. No third-party source artwork is bundled.
